# Desert Vendetta '76

Ein browserbasiertes 3D-Actionspiel, direkt spielbar über GitHub Pages:

https://realtommygreen.github.io/desert-vendetta-76/

Dieses Repository enthält ausschließlich die Web-Version mit ihren Audio-Assets. Die Android-App und ihre Build-Dateien sind nicht enthalten.

Aktueller Spielstand: **Version 1.9**

## Start

Öffne die GitHub-Pages-Adresse in einem aktuellen Desktop- oder Mobilbrowser. Alternativ kann der Ordner über einen lokalen HTTP-Server ausgeliefert werden.
